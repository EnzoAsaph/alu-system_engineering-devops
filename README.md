# alu-system_engineering-devops

System engineering and DevOps projects for ALU.

## Directories

- `ssh` — SSH keys, client configuration and connection scripts
- `web_server` — Nginx installation, redirects, custom error pages and `scp` transfers
- `load_balancer` — custom HTTP headers and HAProxy round-robin load balancing
- `firewall` — UFW firewall rules
- `web_stack_debugging_0` — first web stack debugging exercise (Apache in Docker)
- `web_stack_debugging_1` — Nginx listening on port 80
- `web_stack_debugging_2` — running Nginx as the `nginx` user on port 8080
- `web_stack_debugging_3` — debugging WordPress on LAMP with `strace`, fixed with Puppet
